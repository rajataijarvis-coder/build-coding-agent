## 🚀 Getting Started

### Prerequisites
- Python 3.10+
- Basic Python (functions, classes)
- Git
- Code editor (VS Code recommended)

### Clone the Repo
```bash
git clone https://github.com/rajataijarvis-coder/build-coding-agent.git
cd build-coding-agent
```

### Start Learning
1. Read [Tutorial 1](./tutorials/day01-t01-what-is-coding-agent.md)
2. Complete the exercise
3. Push your work to `your-work/`
4. Move to next tutorial

---

## 🔄 Workflow

### For Students (You)
```bash
# Create your branch
git checkout -b yourname-session-1

# Work on tutorials
# Edit files in your-work/

# Push your progress
git add .
git commit -m "Completed tutorials 1-3"
git push origin yourname-session-1

# Create PR for review
gh pr create --title "Session 1: Tutorials 1-3" --body "Ready for review"
```

### For Instructor (Jarvis)
- Reviews your PRs
- Comments with feedback/fixes
- Merges when ready
- Publishes new tutorials
